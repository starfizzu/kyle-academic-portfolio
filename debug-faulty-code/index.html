sumUpTo — Off-By-One Infinite Loop

Runtime
✕ Faulty Code
function sumUpTo(n) {
  let total = 0;
  for (let i = 1; i = n; i++) {
    // i = n is always truthy!
    total += i;
  }
  return total;
}
✓ Fixed Code
function sumUpTo(n) {
  let total = 0;
  for (let i = 1; i <= n; i++) {
    // <= gives correct exit condition
    total += i;
  }
  return total;
}
Bug: InfiniteLoop — loop condition i = n never becomes false

}
