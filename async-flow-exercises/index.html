JavaScript Event Loop Lab

Synchronous · Macrotasks · Microtasks · Async/Await

Step 1

Synchronous Execution

console.log("A");
console.log("B");
console.log("C");
Output
Running...
Expected: A → B → C
Step 2

setTimeout (Macrotask)

console.log("Start");
setTimeout(() => {
  console.log("Timeout");
}, 0);
console.log("End");
Output
Running...
Expected: Start → End → Timeout
Step 3

Promise (Microtask)

console.log("Start");
Promise.resolve().then(() => {
  console.log("Promise");
});
console.log("End");
Output
Running...
Expected: Start → End → Promise
Step 4

Microtask vs Macrotask

console.log("Start");
setTimeout(() => {
  console.log("Timeout");
}, 0);
Promise.resolve().then(() => {
  console.log("Promise");
});
console.log("End");
Output
Running...
Expected: Start → End → Promise → Timeout
Step 5

Async / Await Flow

async function test() {
  console.log("1");
  await Promise.resolve();
  console.log("2");
}
console.log("3");
test();
console.log("4");
Output
Running...
Expected: 3 → 1 → 4 → 2
Step 6

Advanced Flow Challenge

console.log("A");
setTimeout(() => {
  console.log("B");
}, 0);
Promise.resolve().then(() => {
  console.log("C");
});
console.log("D");
Output
Running...
Expected: A → D → C → B
